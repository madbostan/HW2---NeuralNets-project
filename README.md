# CIFAR-10 Image Classification

Простой проект для обучения модели **сверточной нейронной сети (CNN)** на наборе данных CIFAR-10 и создания интерактивного веб-приложения на **Streamlit** для классификации изображений.

---

## Цель проекта

* Научиться классифицировать изображения с помощью CNN в **PyTorch**.
* Создать веб-приложение для загрузки изображений и получения предсказания.

---

## Структура проекта

```
/cifar10_project
│
├─ cifar10_training.ipynb      # Jupyter-ноутбук с обучением модели
├─ streamlit_app.py            # Streamlit-приложение
├─ cifar_net.pth               # Сохранённая модель
├─ requirements.txt            # Зависимости проекта
└─ README.md                   # Этот файл
```

--- 

## Как запустить

1. Установить зависимости:

```bash
pip install -r requirements.txt
```

2. Запустить Streamlit-приложение:

```bash
streamlit run streamlit_app.py
```

3. Загрузить изображение через веб-интерфейс и увидеть предсказанный класс.

---

## Данные

* **Набор данных:** CIFAR-10
* **Количество классов:** 10
  `plane, car, bird, cat, deer, dog, frog, horse, ship, truck`
* **Размер изображений:** 32x32 пикселя

---

## Результаты

* Точность модели после обучения: ~60% (можно увеличить количеством эпох и настройкой сети)
